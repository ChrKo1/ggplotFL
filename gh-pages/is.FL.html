Methods to determine the class of a given object

Description

These methods return TRUE if the given object is of the corresponding class, and FALSE otherwise.

These methods should be substituted by calls to is and will very likely be deprecated in future releases.

Generic function

is.FLQuants(object)

is.FLBiols(object)

is.FLIndices(object)

is.FLStocks(object)

Describe method

Examples

# This call ...
is.FLQuant(FLQuant())



[1] TRUE



# ... should be substituted by
is(FLQuant(), 'FLQuant')



[1] TRUE
